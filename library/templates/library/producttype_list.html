URL Patterns

Finding information on a specific model is as simple as entering in the correct URL. All URLs in the library module begin with the keyword 'library'. The subsequent keywords will follow a pattern <model type><model family><model number>. For example, if I would like to find information on the MX-3071 I would enter 127.0.0.1:8000/library/mfp/phoenix-two/mx-3071
